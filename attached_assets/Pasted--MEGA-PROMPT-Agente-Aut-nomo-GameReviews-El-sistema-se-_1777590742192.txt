# 🤖 MEGA PROMPT — Agente Autónomo GameReviews
# El sistema se actualiza SOLO cada 24 horas con IA

---

## OBJETIVO
Construir un agente autónomo que corra en segundo plano y actualice GameReviews automáticamente cada día sin que el programador toque nada. La IA decide qué juegos agregar, genera las descripciones, obtiene las imágenes y actualiza la base de datos sola.

---

## LO QUE VA A HACER EL AGENTE CADA 24 HORAS

1. Consultar RAWG API → detectar juegos nuevos o trending de las últimas 24h
2. Consultar ITAD API → detectar juegos que entraron en oferta
3. Pasarle esa data cruda a Claude → Claude genera descripciones, análisis y decide si vale la pena agregar el juego
4. Actualizar automáticamente `data/juegos.json` con los juegos nuevos
5. Crear un archivo `data/noticias.json` con novedades del día
6. Loggear todo en `logs/agente.log` con timestamp

---

## ARQUITECTURA TÉCNICA

### Nuevo archivo: `server/agente.ts`
Este es el cerebro del agente. Debe hacer lo siguiente:

```typescript
// Corre cada 24 horas con node-cron
// Schedule: '0 6 * * *' (6am todos los días)

async function runAgente() {
  // PASO 1: Obtener juegos trending de RAWG
  const trendingGames = await fetchFromRAWG('/games', {
    ordering: '-added',
    dates: `${yesterday},${today}`,
    page_size: 20
  });

  // PASO 2: Para cada juego nuevo, pedirle a Claude que evalúe si vale la pena
  const prompt = `
    Eres el editor de GameReviews, una plataforma de reseñas de videojuegos.
    
    Aquí hay ${trendingGames.length} juegos nuevos/trending de hoy:
    ${JSON.stringify(trendingGames, null, 2)}
    
    Tu tarea:
    1. Selecciona máximo 3 juegos que valgan la pena agregar a la plataforma
    2. Para cada uno genera:
       - descripcion: 2 oraciones atractivas en español
       - descripcionCorta: 1 frase de máximo 10 palabras
       - genero: array de géneros en español
       - plataformas: array de plataformas
       - etiquetas: array de 5 tags relevantes
       - porQueDestaca: 1 oración de por qué este juego es notable
    3. Devuelve SOLO un JSON array con los juegos seleccionados y sus campos generados
    
    Formato de respuesta (SOLO JSON, sin texto extra):
    [
      {
        "nombre": "...",
        "slug": "...",
        "descripcion": "...",
        "descripcionCorta": "...",
        "genero": [...],
        "plataformas": [...],
        "etiquetas": [...],
        "porQueDestaca": "...",
        "imagen": "url de rawg",
        "imagenBanner": "url de rawg",
        "rawgId": 123,
        "metacritic": 85,
        "fechaLanzamiento": "2025-01-01"
      }
    ]
  `;

  const claudeResponse = await fetch('https://api.anthropic.com/v1/messages', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'x-api-key': process.env.ANTHROPIC_API_KEY,
      'anthropic-version': '2023-06-01'
    },
    body: JSON.stringify({
      model: 'claude-opus-4-5',
      max_tokens: 2000,
      messages: [{ role: 'user', content: prompt }]
    })
  });

  // PASO 3: Parsear respuesta y agregar al JSON
  const nuevosJuegos = JSON.parse(claudeResponse.content[0].text);
  
  // PASO 4: Leer juegos.json actual, agregar nuevos, guardar
  const juegosActuales = await readJSON('data/juegos.json');
  const juegosActualizados = [...juegosActuales, ...nuevosJuegos.map((j, i) => ({
    ...j,
    id: juegosActuales.length + i + 1,
    rating: 0,
    reseñas: [],
    precio: 59.99,
    dlcs: [],
    skins: [],
    agregadoPorAgente: true,
    fechaAgregado: new Date().toISOString()
  }))];
  
  await writeJSON('data/juegos.json', juegosActualizados);

  // PASO 5: Generar noticias del día con Claude
  const noticiasPrompt = `
    Basándote en estos juegos trending de hoy: ${JSON.stringify(trendingGames.slice(0, 5))}
    
    Genera 3 noticias cortas de gaming para mostrar en la sección de novedades de GameReviews.
    Cada noticia debe ser relevante, atractiva y en español.
    
    Devuelve SOLO JSON:
    [
      {
        "titulo": "...",
        "resumen": "2 oraciones máximo",
        "categoria": "Lanzamiento|Actualización|Oferta|Comunidad",
        "urgente": false
      }
    ]
  `;

  // ... mismo proceso con Claude para noticias
  
  // PASO 6: Log
  await appendLog(`[${new Date().toISOString()}] Agente completado. ${nuevosJuegos.length} juegos agregados.`);
}
```

## NUEVO ENDPOINT EN server/routes.ts

```typescript
// GET /api/agente/status
// Muestra el log del agente y cuándo corrió por última vez
router.get('/api/agente/status', (req, res) => {
  const log = readFileSync('logs/agente.log', 'utf-8');
  res.json({ 
    ultimaEjecucion: getLastRunTime(),
    proximaEjecucion: getNextRunTime(),
    log: log.split('\n').slice(-20) // últimas 20 líneas
  });
});

// POST /api/agente/run
// Ejecuta el agente manualmente (solo para admin)
router.post('/api/agente/run', async (req, res) => {
  await runAgente();
  res.json({ success: true, mensaje: 'Agente ejecutado manualmente' });
});
```
